# java-utils
Java工具类

## 使用方式
### 采用Maven构建项目
在pom中引入依赖，版本换成对应版本即可,最新版本详见Releases
```
<dependency>
    <groupId>com.jssolo</groupId>
    <artifactId>java-utils</artifactId>
    <version>${java-utils.version}</version>
</dependency>
```
